================================================================================
USAGE INSTRUCTIONS
================================================================================

1. COLOR PALETTE SETUP:
   - Choose a palette from above
   - In Django Admin, go to Branding > Color Palettes
   - Create/Edit the ColorPalette for your tenant
   - Copy-paste all 18 color values from the palette above
   - Save

2. FONT PAIR SETUP:
   - Choose a font pair from above
   - In Django Admin, go to Branding > Font Pairs
   - Create/Edit the FontPair for your tenant
   - Copy-paste all 4 font values from the pair above
   - Save

3. PREVIEW:
   - Open BRANDING_PRESETS_FULL.html in a browser to see all themes live
   - Changes reflect immediately in your templates via CSS variables

4. CUSTOMIZATION:
   - Mix and match colors from different palettes
   - Adjust hex values to match your brand
   - Test with success/warning/error states
   - Ensure sufficient color contrast for accessibility


================================================================================
TECHNICAL NOTES
================================================================================

- All colors use hex format (#RRGGBB) except primary_glow (rgba)
- Font weights are comma-separated (e.g., "400,600,700")
- Google Fonts are automatically loaded via FontPair.google_fonts_url()
- CSS variables update dynamically without page refresh
- Dark mode overrides: dark_primary, dark_surface, dark_text (optional)


================================================================================
COLOR PALETTE FIELD MAPPING
================================================================================

Primary Colors (Brand Identity):
    primary         → Main brand color (buttons, links, CTAs)
    primary_dark    → Hover states, darker sections
    primary_light   → Light backgrounds, subtle accents
    primary_glow    → Box shadows, glow effects (rgba format)

Secondary Colors (Supporting):
    secondary       → Headers, footers, navigation
    secondary_light → Lighter navigation sections

Accent Colors:
    accent          → Highlights, special elements, icons

Surface Colors (Backgrounds):
    surface         → Main page background
    surface_alt     → Card backgrounds, alternate sections

Text Colors:
    text            → Primary text (headings, body)
    text_muted      → Secondary text (captions, metadata)

Footer Colors:
    footer_bg       → Footer background
    footer_text     → Footer text color

Border Colors:
    border          → Borders, dividers, separators

Status Colors (UI Feedback):
    success         → Success messages, confirmations
    warning         → Warnings, pending states
    error           → Errors, rejections, alerts
